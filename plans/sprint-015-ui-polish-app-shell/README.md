# Sprint 015 - UI Polish App Shell

## Goal

Turn the frontend from a single-column MVP form into a scalable video workflow app shell.

## Scope

- Add a sidebar-based app layout.
- Redesign the generate page into a two-column workspace.
- Redesign the job detail page with clearer status, actions, logs, and output panels.
- Add placeholder routes for upcoming Jobs, Editor, and Settings work.
- Keep backend behavior unchanged.

## Done Criteria

- Navigation is visible on desktop and usable on mobile.
- Generate and job detail pages no longer depend on a narrow single-column layout.
- UI text remains English.
- Frontend typecheck and build pass.
